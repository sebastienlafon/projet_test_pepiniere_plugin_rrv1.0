== Description ==
Reservation Booking is designed for the unique reservation requirements of 
a Room or any service. It is the only wordpress plugin that allows guests 
to book a room. 

 
== Installation ==

Upload from the WordPress Dashboard

1. From the dashboard of your site, go to "Plugins" on the left sidebar and click "Add New"
2. Select the Upload tab and click "Choose File"
3. Find the zip file on your computer and click "Open" - Now click "Install Now"
4. When it has finished installing select Yes to activate the plugin
5. You will now see "Reservation Booking" in the left sidebar in the dashboard - click to see your new booking calendar interface!


Upload Using FTP

1. Upload the folder "Reserve Room" to the `/wp-content/plugins/` directory.
2. Now go to the plugins page in the dashboard to activate the plugin
